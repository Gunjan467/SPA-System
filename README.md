# SPA-System (Student Performance Analysis System)

The **Student Performance Analysis System** is a web-based application developed using **Python (Flask)**, **MySQL**, and **HTML**.  
This project focuses on analyzing and managing student academic performance through attendance, marks, GPA calculation, and progress tracking.

## Features
- User authentication with role-based access (Student / Teacher / Admin)
- Student dashboard with attendance, GPA, marks, and progress analysis
- Teacher dashboard for managing attendance, marks, and feedback
- Semester-wise GPA calculation
- Subject-wise performance comparison
- Class-level analytics and insights
- Feedback system between teachers and students

## Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS
- **Database:** MySQL

### Purpose
This project was built for **learning and practice**, focusing on backend development, database design, and real-world CRUD operations.
